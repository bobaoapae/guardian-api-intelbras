# Changelog

## [1.0.0] - 2024-01-25

### Added
- Initial release
- OAuth 2.0 authentication with Intelbras Cloud
- ISECNet protocol implementation for direct panel communication
- Real-time status sync (arm/disarm, zones, triggers)
- Zone management with custom friendly names
- Device password storage for auto-sync
- Support for alarm panels (AMT, ANM series)
- Support for electric fences (ELC series)
- Web UI for testing and configuration
- REST API endpoints for Home Assistant integration
